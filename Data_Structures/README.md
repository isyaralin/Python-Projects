# Core Data Structures & Applications

This directory contains a collection of fundamental and custom data structures, alongside their practical applications, implemented entirely in Python. 

Throughout my academic year, I dedicated significant time to studying algorithmic complexity and building these structures from scratch. This hands-on practice gave me a strong foundation in understanding how they work under the hood, allowing me to optimize their efficiency over multiple iterations. 

Beyond just the raw implementations, I have also applied these structures to various daily programming problems to design effective, correct, and highly optimized solutions.
