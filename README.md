# Python Engineering & AI Portfolio

A collection of Python projects focusing on Artificial Intelligence, Constraint Satisfaction (SAT/CSP), Graph Theory, and Software Architecture.

---

## Technical Highlights & Domains

### 1. Artificial Intelligence & Heuristic Search
* **A* Search & Monotone Heuristic Engineering:** Custom consistent heuristics designed for pathfinding over infinite 2D and 3D grids, including diagonal, rook, and jumper movement constraints.

### 2. Constraint Satisfaction & Combinatorial Optimization
* **NP-Complete Problem Solving:** Formulated boolean satisfiability (SAT) encodings for the 3-Partition problem using `python-sat`.
* **Graph Total Coloring:** Implemented CSP algorithms via `python-constraint` and `networkx` to determine the total chromatic number of arbitrary undirected graphs.

### 3. Object-Oriented Design & Applied Python
* **Clean Code & Testing:** End-to-end Python implementations built with strict performance constraints (memory profiling and runtime limits) paired with unit test suites.
